Cloud Native Kansai#01にいってきました

Kubernetes使ってますか？というのをテーマに、Cloud Nativeなお話を聞ける会でした。

課題を共感したい

と、思いました。

関西でKubernetesの導入事例がまだまだ少ない中で、個人的に聞きたいのはやはり本番導入の事例、特に実際使ってみて課題に感じたところやハマったところおよびその対策だったり、Kubernetesを導入したことで現状の課題が解決されて幸せになったねといったお話。そういう話がいっぱい出てくることで、導入までのハードルが下がったり、同じ課題を抱えている人がKubernetes導入してみようかな、という気持ちになったりして、結果的に関西でもCloud Native界隈が盛り上がるのではないかなぁと。

あとはもうちょっと課題感を共感したい。僕の想像力が足りないだけかもですけど、課題がさらっと言われてしまうケースが多くて、Kubernetesを導入して何がどう幸せになったのか、という話があんまりピンと来ない。共感できてない。決して初心者向けのお話というわけでもないと思うんですけどどうでしょう。

単純に流行ってるからk8sでやろうぜ、っていうのも全然いいんですけど、別にEC2とRDSとELBでええやん、になりがち。適材適所、課題あってこその技術。そういう意味でちゃんとその点に切り込んでいたxxさんの発表はよかったと思う。

ベンダーさんの話は自社製品のお話になるけど、そこもやっぱりただ紹介してるだけじゃ意味なくて課題をもっとはっきりして欲しい。で、何が嬉しいの？になってしまう。

とはいえ、ベンダーさんやら導入してみましたっていう人やら実際使っててこんな課題ありました、みたいなお話まで聞けるのは関西では貴重な機会だと思うのでまた行きます。
