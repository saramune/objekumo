EKS workshop挑戦記

EKS workshopというものがあるので、やってみました。
https://eksworkshop.com/

Introduction

Kubernetesとは、EKSとは。読むだけ。EKSの内部の仕組みが書いてあるのがよかった。（ドキュメント見たら書いてるんだろうけど。

Start the workshop…(所要時間：10分)

  • Admin権限のIAMユーザ作成
  • Cloud9構築
  • cliのインストール、ソースの取得

Launch using eksctl(所要時間：20分)

  • eksctlのインストール
  • EKSクラスタの構築（ここだけで15分くらい待つ）

Deploy the Kubernetes Dashboard(所要時間：5分)

  • Dashboardのデプロイ

Deploy the Example Microservices(所要時間：20分)

  • サンプルのmicroservice(nodeのバックエンド, crystalのバックエンド, frontend)
  • 各podがどのノードにあって、っていうのが可視化されるアプリなので、podをスケールアウトするとその様子がみてとれる。よくできてる。

Helm(所要時間：35分)

  • helmを使ってnginxをデプロイ
  • helmを使って先程のmicroserviceをデプロイ、ロールバック

Health Checks(所要時間：15分)

  • liveness probe(podの死活)とreadiness probe(podへのトラフィック送信可否)について

Autoscaling our Applications and Clusters(所要時間：20分)

  • podのAutoScaling、nodeのAutoScalingについて
  • 負荷をかけて実際にスケールアウトすることを確認する
